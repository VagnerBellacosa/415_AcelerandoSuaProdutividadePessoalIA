# ☕💣 PassoAPasso.md

# Desafio Criativo: Acelerando Sua Produtividade Pessoal com IA

> "Um prompt mal definido é como um JCL sem PARM: até executa, mas ninguém sabe exatamente o que vai acontecer."

------

# 🎯 Objetivo do Desafio

Este desafio tem como objetivo ensinar os fundamentos da criação de prompts para Inteligência Artificial.

Na prática, você aprenderá a:

- Definir uma intenção clara.
- Fornecer contexto relevante.
- Adicionar restrições.
- Construir um prompt completo.
- Publicar sua solução.
- Compartilhar sua entrega na plataforma DIO.

Embora pareça simples, este exercício apresenta um dos conceitos mais importantes da IA Generativa:

**A qualidade da resposta depende diretamente da qualidade da pergunta.**

------

# 🧠 Antes de Começar

Imagine que a IA é um novo operador de computador recém-contratado.

Se você disser apenas:

```text
Faça um relatório.
```

O resultado provavelmente será genérico.

Mas se disser:

```text
Crie um relatório para gestores de TI sobre redução de custos em ambiente Mainframe utilizando IA.
```

A resposta será muito mais próxima do esperado.

É exatamente isso que este desafio ensina.

------

# 🧱 Etapa 1 — Defina a Intenção

## O que fazer?

Defina claramente:

- O que deseja gerar.
- Para quem.
- Qual objetivo.

## Modelo

```text
Quero que a IA gere [conteúdo]
para [público]
com o objetivo de [resultado].
```

## Exemplo Tradicional

```text
Quero que a IA gere uma lista de tarefas para estudantes com o objetivo de melhorar a organização dos estudos.
```

## Exemplo Bellacosa Mainframe

```text
Quero que a IA gere um plano de estudos para profissionais Mainframe com o objetivo de acelerar a capacitação em COBOL e z/OS.
```

------

# ✅ Resultado Esperado da Etapa 1

Você deve possuir uma frase clara contendo:

| Elemento | Definido |
| -------- | -------- |
| Conteúdo | Sim      |
| Público  | Sim      |
| Objetivo | Sim      |

------

# 🧱 Etapa 2 — Adicione Contexto

## O que fazer?

Agora informe à IA informações adicionais.

Pense:

- Em qual cenário isso será utilizado?
- Existe alguma característica específica?
- Há alguma limitação?

## Modelo

```text
Considere o seguinte contexto:
[contexto]
```

## Exemplo

```text
Considere o seguinte contexto:
profissionais que trabalham com COBOL, JCL, CICS e DB2 em ambiente IBM Mainframe.
```

------

# 💡 Dica Bellacosa

Uma IA sem contexto é como procurar um dataset sem saber o HLQ.

Quanto mais contexto você fornecer:

- Melhor precisão.
- Menos alucinações.
- Menos retrabalho.

------

# 🧱 Etapa 3 — Defina o Formato

Agora determine como deseja receber a resposta.

## Exemplos

### Lista

```text
O conteúdo deve ser apresentado em formato de lista.
```

### Tabela

```text
O conteúdo deve ser apresentado em formato de tabela.
```

### Roadmap

```text
O conteúdo deve ser apresentado como um roadmap semanal.
```

### Tutorial

```text
O conteúdo deve ser apresentado como um passo a passo.
```

------

# 💡 Sugestão Profissional

Sempre peça:

```text
Utilize títulos e subtítulos.
```

e

```text
Organize em tópicos.
```

Isso melhora significativamente a legibilidade.

------

# 🧱 Etapa 4 — Defina Restrições

Agora informe o que NÃO deseja.

## Exemplos

```text
Evite linguagem técnica.
Evite respostas muito longas.
Evite informações genéricas.
Evite conceitos sem aplicação prática.
```

------

# 💡 Dica de Engenharia de Prompt

Muitos usuários dizem apenas:

```text
Faça isso.
```

Os melhores resultados surgem quando também dizemos:

```text
Não faça aquilo.
```

------

# 🧱 Etapa 5 — Monte o Prompt Final

Agora una todas as partes.

## Estrutura

```text
Quero que a IA gere [conteúdo]
para [público]
com o objetivo de [resultado].

Considere o seguinte contexto:
[contexto].

O conteúdo deve ter formato de [formato].

Evite [restrições].
```

------

# 🎯 Exemplo de Solução Completa

```text
Quero que a IA gere um plano de estudos para profissionais Mainframe IBM z/OS, com o objetivo de acelerar a capacitação técnica.

Considere o seguinte contexto:
profissionais que desejam aprender COBOL, JCL, CICS, DB2, RACF e conceitos modernos de integração com APIs.

O conteúdo deve ser apresentado como roadmap semanal contendo objetivos, exercícios, laboratórios e indicadores de progresso.

Evite explicações excessivamente acadêmicas e conceitos sem aplicação prática.
```

------

# 📁 Etapa 6 — Criar o Arquivo da Entrega

Salve o arquivo.
