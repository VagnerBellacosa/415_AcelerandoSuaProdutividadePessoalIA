# 🚀 Desafio Criativo: Acelerando Sua Produtividade Pessoal com IA

## 📖 Sobre o Desafio

Este desafio tem como objetivo ensinar os conceitos fundamentais de **Prompt Engineering**, demonstrando como criar instruções claras e eficientes para ferramentas de Inteligência Artificial.

Ao longo da atividade, o participante aprenderá a estruturar um prompt utilizando três componentes principais:

- 🎯 Intenção
- 📝 Contexto
- ⚙️ Restrições e formato

Ao final, todas as informações serão combinadas em um único prompt capaz de gerar respostas mais precisas, úteis e alinhadas ao resultado desejado.

------

# 🎯 Objetivos de Aprendizagem

Ao concluir este desafio você será capaz de:

- Entender o que é um prompt.
- Aprender boas práticas de comunicação com IA.
- Organizar ideias de forma estruturada.
- Criar instruções mais eficientes para modelos de IA.
- Compartilhar seu trabalho utilizando plataformas de hospedagem de arquivos.

------

# 🧱 Estrutura do Desafio

O exercício está dividido em três etapas simples.

## Passo 1 — Defina a Intenção

Nesta etapa você deve responder:

- O que deseja que a IA faça?
- Para quem será o conteúdo?
- Qual resultado espera obter?

### Modelo

```text
Quero que a IA gere [tipo de conteúdo] para [público], com o objetivo de [resultado].
```

### Exemplo

```text
Quero que a IA gere uma lista de tarefas para estudantes, com o objetivo de melhorar a organização dos estudos.
```

------

## Passo 2 — Adicione Contexto e Restrições

Agora você deverá detalhar como a resposta deve ser construída.

### Perguntas para ajudar

- Qual contexto deve ser considerado?
- Existe um formato específico?
- Há algo que deve ser evitado?

### Modelo

```text
Considere o seguinte contexto: [contexto].

O conteúdo deve ter [formato].

Evite [o que evitar].
```

### Exemplo

```text
Considere o seguinte contexto: rotina de estudos diária.

O conteúdo deve ter formato de lista simples.

Evite linguagem técnica.
```

------

## Passo 3 — Monte o Prompt Final

Chegou o momento de unir tudo em uma única instrução.

### Modelo

```text
Quero que a IA gere [tipo de conteúdo] para [público], com o objetivo de [resultado].

Considere o seguinte contexto: [contexto].

O conteúdo deve ter [formato].

Evite [o que evitar].
```

### Exemplo Completo

```text
Quero que a IA gere uma lista de tarefas para estudantes, com o objetivo de melhorar a organização dos estudos.

Considere o seguinte contexto: rotina de estudos diária.

O conteúdo deve ter formato de lista simples.

Evite linguagem técnica.
```

# 💡 Dicas para Criar Bons Prompts

## Seja específico

Em vez de:

```text
Crie um texto sobre tecnologia.
```

Prefira:

```text
Crie um artigo de 500 palavras sobre Inteligência Artificial para iniciantes.
```

## Defina o público

Exemplo:

```text
Para estudantes.
Para profissionais.
Para iniciantes.
Para gestores.
```

## Defina o formato

Exemplos:

- Lista
- Tabela
- Artigo
- Tutorial
- Roadmap
- Resumo Executivo

## Informe o que evitar

Exemplos:

- Linguagem técnica
- Termos complexos
- Conteúdo muito extenso
- Jargões corporativos

# 📁 Como Criar o Arquivo da Entrega

Crie um arquivo chamado:

```text
PromptFinal.md
```

ou

```text
PromptFinal.txt
```

Cole seu prompt final dentro dele e salve.

------

# ☁️ Como Entregar o Projeto

A DIO aceita diferentes formas de hospedagem.

## Opção 1 — Google Drive

1. Faça upload do arquivo.
2. Clique em Compartilhar.
3. Selecione:
   - Qualquer pessoa com o link.
4. Copie o link.
5. Cole o link na área de submissão da DIO.

------

## Opção 2 — GitHub

### Criar Repositório

1. Acesse GitHub.
2. Clique em New Repository.
3. Escolha um nome.

Exemplo:

```text
desafio-produtividade-com-ia
```

1. Marque como Público.
2. Crie o repositório.

### Enviar Arquivo

1. Clique em Add File.
2. Upload Files.
3. Envie o arquivo PromptFinal.md.
4. Faça o Commit.

### Obter o Link

1. Abra o arquivo enviado.
2. Copie a URL do navegador.
3. Cole na plataforma da DIO.
